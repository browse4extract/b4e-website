# Assets Folder

Place your Browse4Extract visual assets here:

- `logo.png` - Main application logo (app_image.png)
- `hero-screenshot.png` - Main interface screenshot for hero section
- `element-picker.png` - Visual Element Picker in action
- `settings.png` - Settings/modal screenshot
- `results.png` - Extraction results screenshot
- `export-formats.png` - JSON/CSV/Excel export examples
- `profiles.png` - Profile management screenshot

All images will be accessible via `/images/filename.png` in the app.
